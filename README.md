# Olympiad_preparing_platform
Olympiad Preparing Platform — это современная образовательная веб-платформа, разработанная для эффективной подготовки к олимпиадам по программированию и другим техническим дисциплинам. Платформа объединяет в себе классические методы обучения с инновационными игровыми механиками, создавая увлекательную и результативную среду для студентов и школьников.

Проект предоставляет пользователям возможность не только решать задачи в индивидуальном тренировочном режиме, но и участвовать в соревнованиях PvP (игрок против игрока), где они могут состязаться с другими участниками в реальном времени. Система автоматически рассчитывает рейтинг Elo после каждого матча, позволяя объективно оценивать уровень подготовки и отслеживать прогресс.

Ключевой особенностью платформы является поддержка различных типов задач: от классических алгоритмических задач по программированию с автоматической проверкой кода Python до теоретических задач по математике, физике и другим предметам с текстовой проверкой ответов. Это делает платформу универсальным инструментом для комплексной подготовки к олимпиадам.

Администраторы получают полноценную панель управления, позволяющую добавлять новые задачи, настраивать тесты, управлять пользователями и контролировать ход соревнований. Для пользователей доступна подробная статистика с историей всех решений, показателями успешности и визуализацией прогресса обучения.

Платформа реализована на современном технологическом стеке: Python 3.8+ с использованием микрофреймворка Flask для бэкенда, SQLAlchemy ORM для работы с базой данных SQLite, Flask-Login для аутентификации и Flask-SocketIO для организации реального взаимодействия между пользователями в PvP-режиме.

## Простой способ установки

Olympiad Preparing Platform разработана с акцентом на простоту установки и запуска. 
Для начала работы вам не потребуется сложная настройка серверов или установка дополнительного программного обеспечения.

Шаги для быстрого запуска:
1. Скачайте архив проекта с GitHub и распакуйте его в удобную директорию на вашем компьютере.

2. Установите необходимые библиотеки Python, выполнив в командной строке:
    pip install flask flask-login flask-socketio sqlalchemy werkzeug
Это установит все основные зависимости, необходимые для работы платформы.

3. Откройте проект в вашем редакторе кода (например, VS Code, PyCharm или любой другой).

4. Наша тестирующая система работает коррекроктно только на опреционной системе Windows или Linux. В случае работы на macOS для корректной работы тестирующей системы в строках 952 и 1065 в файле main.py надо заменить 'python' на 'python3'.

5. ! так как API-ключ является секретной информацией, то для работы моделя искуственного интеллекта в файл ai.py надо вставить его в строку 4 в переменную API_KEY: 'VN2vu0EfT_5grsmJLtP8jhNq-FI1UiX8_eO-ND' 

6. Запустите главный файл приложения main.py прямо из редактора

7. После этих простых шагов платформа будет доступна по адресу http://127.0.0.1:8080.
Вы можете сразу приступать к регистрации и использованию всех функций.
